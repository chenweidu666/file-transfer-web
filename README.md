# 文件上传服务器

通过手机浏览器上传文件到电脑，无需安装任何应用。

## 使用方法

```bash
python3 upload_server.py
```

然后在手机浏览器中访问终端显示的地址即可上传文件。

## 功能

- 📱 手机浏览器直接访问上传
- 📁 支持多文件上传
- 🎨 美观的移动端界面
- 🔒 仅限局域网访问
- 📊 显示已上传文件列表

## 配置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-p, --port` | 端口号 | 8000 |
| `--host` | 绑定地址 | 0.0.0.0 |

## 示例

```bash
# 默认端口
python3 upload_server.py

# 自定义端口
python3 upload_server.py -p 9000
```
