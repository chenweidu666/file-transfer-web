# Screenshots

This directory contains screenshots of the File Transfer Web interface.

## How to Capture Screenshots

Since this is a web-based file transfer tool, you can capture screenshots when the server is running:

1. Start the server:
   ```bash
   ./server.sh start
   ```

2. Access the web interface from a browser (either on the same machine or from another device on the same network):
   - The server will display the URL when started, typically something like `http://192.168.x.x:8000`

3. Take screenshots of:
   - Main upload interface
   - Upload progress
   - File history list
   - Mobile view (when accessed from a mobile device)

## Typical Interface Elements

The interface includes:
- Drag-and-drop upload zone
- File selection button
- Real-time upload progress bars
- File history with timestamps and sizes
- Download and delete controls
- Responsive design for both desktop and mobile

To contribute screenshots, simply place them in this directory with descriptive names (e.g., `upload-interface.png`, `mobile-view.png`).
